# Coupon-System

1. 멀티 모듈 구현

3. 스트레스 테스트 with Locust

*Aws EC2, Aws RDS, Aws Elastic Cache,*

*Java 17, Spring Boot 3.1, Spring Mvc, JPA, QueryDsl*

***Database***

*Mysql, Redis, H2*

***Monitoring***

*Aws Cloud Watch, Spring Actuator, Promethous, Grafana*

***Etc***

*Locust, Gradle, Docker*
